# COMTER
## ✅프로젝트 소개
<div>Comter 프로젝트 이후 Js DOM 사용의 미숙함과 함수를 만드는 것이 익숙치 않아 이번 프로젝트에서 DOM과 함수를 연습하고 자 하였다.</div>
<div>해당 프로젝트에서는 Manager페이지에서 입력한 값을 Main페이지에 띄워 사용자가 해당 아이템을 선택하고 장바구니 기능을 통해 값을 넣고자 하였다.</div>
